# Claim Timeout

Claim a specific expired channel

<figure><img src="../../../../.gitbook/assets/Screenshot 2024-08-17 at 6.35.52â€¯PM.png" alt=""><figcaption><p>Claim Timeout Page</p></figcaption></figure>

```bash
# Format of the command in the SNET CLI

snet channel claim-timeout [-h] [--multipartyescrow-at MULTIPARTYESCROW_AT]
                           [--gas-price GAS_PRICE]
                           [--wallet-index WALLET_INDEX] [--yes]
                           [--quiet | --verbose]
                           CHANNEL_ID
```

User flow:

* Input the channel ID
* Input any optional parameters you would like
* Click the "Claim Timeout" button
